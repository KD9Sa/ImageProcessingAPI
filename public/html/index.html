Image Processing API

The API is created to perform some handful processes on images applying the latest and fastest technologies available in the market for processing images. Currently the API only supports resizing images but we are planning to add more features soon.


Get Started

Processing images is available through the parameters provided after /api. There are currently 3 main parameters:

  • imageName: It is a mandatory parameter. It accepts the name of any image available in public/imgs/original folder.
  • width: It is an optional parameter. used if you desire to resize an image.
  • height: It is an optional parameter also. Used along with 'width' parameter to resize an image.

If imageName was provided alone, the original image will be displayed.
Example:
/api?imageName=fjord

If imageName was provided along with width and height parameters, the image will be resized and cached to thumbnails folder.
Example:
/api?imageName=fjord&width=500&height=400
